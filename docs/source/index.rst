Welcome to ESSA documentation!
==================================

ESSA (Easy Singular Spectrum Analysis) is a Python package for performing Singular Spectrum Analysis (SSA) on time series data.

   :maxdepth: 2
   :caption: Contents:

Key Features
-----------

* Support for both basic SSA and Toeplitz SSA methods
* Full and randomized SVD options for large datasets
* Efficient decomposition and reconstruction of time series
* Intuitive API for time series analysis
